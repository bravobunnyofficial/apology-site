# 🙏 Letters For You

An emotional and interactive apology website that expresses 
deep feelings through animated love letters and heartfelt messages.

🌐 **Live Site** — [Visit Letters For You](https://an-apology-letter.netlify.app/)

---

## 💌 About This Project

Sometimes words are hard to say out loud. This website says them 
beautifully — through animated envelopes, memory cards, typewriter 
text, and a glowing final love message.

---

## ✨ How It Works

1. Opening screen with a floating envelope stack
2. Click to enter — 4 envelopes fly in from different directions
3. Each envelope opens with an animation revealing a memory card
4. Memory card shows an image and typewriter text effect
5. After all 4 letters are read — a final apology message appears
6. Click "Stay with me" — a glowing **I LOVE YOU** poster appears

---

## 🎯 Features

- 💌 4 animated envelopes with fly-in effect
- ✍️ Typewriter text effect on each memory card
- 🌸 Floating hearts and background heart animations
- 💖 Glowing final love poster with pulse animation
- 🎵 Background music
- 📱 Fully mobile responsive
